# Dewey

Dewey is a terminal-based bullet journal that lets you catalog your days and weeks as simply or as intricately as you like.
## Features

- **Daily Log**: Jot down what the day has in store, with different indicators for tasks, events and notes.
- **Monthly View**: Monthly calendar that lets you jump around and plan ahead.

## Using Dewey
*Using a BASH based terminal on Linux or Mac*

Before downloading, make sure .NET Core 8.0 or later is installed. After installing, download Dewey as a zip file using the green 'Code' button above. Then, extract the file and navigate to the resulting folder. In the program folder, launch Dewey using:
```bash
dotnet run
```

## Future Updates

- **Journal**: A password protected space to jot down how your're feeling in the moment.
- **Kanban**: Creating events will give you the option to add a Kanban board to delegate which subtasks are in the works, in progress, or have been completed.
- **Habit Tracking**: To make sure the healthy ones stick around while the bad ones kick the bucket.
